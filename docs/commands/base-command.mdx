# `processcube`

Um alle verfügbaren Kommandos aufzulisten, kann entweder `processcube` oder `pc` ausgeführt werden.

## Verwendung

```shell
processcube --help
```

:::info
Zur leichteren Lesbarkeit wird in dieser Referenz stets die kurze Variante `pc` genutzt.
:::

```shell
pc --help
```
