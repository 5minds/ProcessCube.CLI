# `pc retry-process-instance`

## Verwendung

```shell
pc retry-process-instance [processInstanceIds...] [options]
pc retry [processInstanceIds...] [options]
```

## Beschreibung

Startet fehlgeschlagene Prozessinstanzen mit den gegebenen Prozessinstanz-IDs auf der verbundenen Engine an der Stelle neu, an der sie fehlgeschlagen sind.

## Argumente

| Name               | Obligatorisch | Beschreibung                         |
| ------------------ | ------------- | ------------------------------------ |
| processInstanceIds | Ja            | IDs der zu stoppenden Prozessmodells |

## Optionen

| Name, Kürzel             | Default | Beschreibung                                                                    |
| ------------------------ | ------- | ------------------------------------------------------------------------------- |
| --output, -o | text    | Steuert das Ausgabeformat: `text` oder `json` |

## Beispiele

### Kurzform

Eine Prozessinstanz kann über ihre ID neugestartet werden:

```shell
pc retry-process-instance 56a89c11-ee0d-4539-b4cb-84a0339262fd
```

Alternativ kann die Kurzform `start` genutzt werden:

```shell
pc retry 56a89c11-ee0d-4539-b4cb-84a0339262fd
```
