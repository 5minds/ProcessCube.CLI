# Installation

## Installationsvoraussetzungen

Die ProcessCube CLI muss per [NPM](https://docs.npmjs.com/) installiert werden.
Hierzu muss [NodeJS](https://nodejs.org/) vorinstalliert sein.

## Erstinstallation

Um die **aktuellste Version** zu installieren, kann folgender Befehl verwendet werden:

```shell
npm install --global @5minds/processcube_cli
```

Um eine **exakte Version** zu installieren, kann diese ebenfalls angegeben werden:

```shell
npm install --global @5minds/processcube_cli@1.0.0
```

## Aktualisieren auf eine andere Version

Zum Aktualisieren der ProcessCube CLI können die zur Erstinstallation genutzten Befehle erneut ausgeführt werden.
